# Lecture 2: Developing on Databricks

### 🔍 What is this?
**Databricks** is a unified data platform in the cloud. Instead of downloading data to your laptop, installing random software packages, and running out of computer memory, Databricks gives your team a shared space in the cloud. It features collaborative notebooks (similar to Google Docs for code) backed by massive, scalable virtual computers called **compute clusters**.

### 💡 Real-World Analogy
Imagine you and your team are trying to build a massive skyscraper, but you are all working from small, isolated home workshops. **Databricks** is like a giant, state-of-the-art construction site where everyone works together side-by-side, with instant access to heavy machinery (compute clusters) that you can rent by the minute.

### 🔑 Key Pillars
* **Compute Clusters:** Groups of cloud computers working together. You can spin up a supercomputer for heavy lifting and turn it off when you're done.
* **Collaborative Notebooks:** Live document interfaces where multiple data scientists can write and execute Python or SQL code simultaneously.
* **Delta Lake:** A modern storage format built into Databricks that keeps your data tables clean, organized, and version-controlled.
